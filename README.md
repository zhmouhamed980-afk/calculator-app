# Calculator App

This is a simple calculator app created using HTML, CSS, and JavaScript.

## How to Use

1. Clone the repository:
   ```bash
   git clone https://github.com/zhmouhamed980-afk/calculator-app.git
   ```
2. Open the `index.html` file in your web browser.
3. Use the calculator buttons to perform calculations.

## Features

- Basic arithmetic operations: addition, subtraction, multiplication, and division.
- Clear button to reset the display.
- Sign toggle button to change the sign of the number.
- Percentage button to calculate the percentage of a number.
- Responsive design that adapts to different screen sizes.
